Projects

Nonsense In Quiz Form

Nonsense In Quiz Form is my simple quiz app created with HTML, JavaScript, and jQuery. The contents of the quiz are imported from a json file that's hosted on a server. This makes modifying and outright changing the whole quiz as simple as modifying the json file with the new content.

Find Fun!

Find Fun! is a web application that takes Google's Geocoding and Places API and marries them together in order to create a tool that helps users find things around their area of interest. The application takes any address, zipcode, or general location (e.g. Walmart in London) and serves up locations of interest to the user based on what they want to find.

D&D Character Creator

The D&D Character Creator is a web application that I built using React and Node.js. The whole project was my attempt at streamlining the character creation process for the game, which was often times tedious and confusing. It uses data taken from a SQL database hosted with Heroku Postgres, which also hosts the Node.js endpoints. The frontend is built on React and hosted with Now.
